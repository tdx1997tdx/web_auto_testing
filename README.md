# web端端UI测试框架
selenium+pytest,POM设计模式+数据驱动

模拟业务：
- 注册登录
- 搜索
## 安装准备
1. 下载对应版本的webdriver并安装
```
https://registry.npmmirror.com/binary.html?path=chromedriver/
```
2. 添加环境变量
3. allure安装：https://blog.csdn.net/redrose2100/article/details/118254838
4. 启动程序
